# Cognitive Thought Engine

**Building the WHY governance layer for autonomous AI agents.**

Most AI agent frameworks answer two questions:  
**WHO** is authorized to act. **HOW** behavior is enforced at runtime.  
We build the third layer — **WHY** an agent does what it does.

| Tier | Question | Examples | Gap |
|------|----------|----------|-----|
| **WHO** | Is this agent authorized? | Azure IAM, Okta, Glasswing | Authorized agents still make unconstitutional decisions |
| **HOW** | Is this action policy-compliant? | Microsoft AGT, NeMo Guardrails, LangChain | Policy-compliant decisions can still be wrong |
| **WHY** | Does this align with constitutional principles? | **constitutional-agent** | — |
